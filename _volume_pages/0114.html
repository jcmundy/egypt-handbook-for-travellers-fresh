{% raw %}CIV{% endraw %}
{% raw %}V. Hieroglyphics.{% endraw %}
{% raw %}By Professor G. Steindorff of Leipsic.{% endraw %}
{% raw %}Repeated attempts were made during the 17th and 18th centuries{% endraw %}
{% raw %}to decipher the peculiar picture-writing of the ancient Egyptians,{% endraw %}
{% raw %}the learned Jesuit father Athanasius Kircher (1601-80) being{% endraw %}
{% raw %}among the earliest to take up the subject. It was not, however,{% endraw %}
{% raw %}until the beginning of the 19th century that the key was found,{% endraw %}
{% raw %}though Sacy, a Frenchman, and Akerblad, a Swede, had previously{% endraw %}
{% raw %}attained a certain amount of success in their efforts to find the clue.{% endraw %}
{% raw %}Dr. Thomas Young, an English scholar, and Francois Champollion,{% endraw %}
{% raw %}a Frenchman, succeeded about the same date (viz. in 1819 and in{% endraw %}
{% raw %}1822) in discovering the long-sought alphabet from a careful com¬{% endraw %}
{% raw %}parison of royal cartouches. Champollion afterwards followed up his{% endraw %}
{% raw %}initial discovery withsuch success that he may fairly rank as the real{% endraw %}
{% raw %}interpreter of the Egyptian hieroglyphics.{% endraw %}
{% raw %}The first clue was afforded by the famous 'Rosetta Stone' (now in the{% endraw %}
{% raw %}British Museum), discovered in 1799 in the Fort St. Julien at Rosetta.{% endraw %}
{% raw %}This tablet of basalt bears three inscriptions: one in the ancient Egyptian{% endraw %}
{% raw %}language, written in hieroglyphics, one in the popular language of a{% endraw %}
{% raw %}later period inscribed in demotic characters, and a third in Greek; but{% endraw %}
{% raw %}the two last are merely translations of the first. The subject of the triple{% endraw %}
{% raw %}inscription is a decree of the Egyptian priests issued in 196 B.C. in honour{% endraw %}
{% raw %}of Ptolemy V. Epiphanes. The first step towards deciphering the hiero¬{% endraw %}
{% raw %}glyphics was made when it was ascertained that the frequently occurring{% endraw %}
{% raw %}groups of signs each enclosed in an oval (so-called cartouche) were the{% endraw %}
{% raw %}names of kings and that the name of Ptolemy must be found among them.{% endraw %}
{% raw %}Champollion and his successors established the phonetic signi¬{% endraw %}
{% raw %}fication of a large number of hieroglyphic symbols, and it then be¬{% endraw %}
{% raw %}came possible, from a knowledge of Coptic, the latest form of the{% endraw %}
{% raw %}ancient Egyptian language, not only to read but also to interpret{% endraw %}
{% raw %}the inscriptions. H. Brugsch, who led the way to the complete{% endraw %}
{% raw %}interpretation of demotic texts, was also the first to point out that{% endraw %}
{% raw %}in hieroglyphic writing, as in the Semitic systems of writing, only{% endraw %}
{% raw %}the consonants were inscribed, while the vowels were omitted as{% endraw %}
{% raw %}not essential.{% endraw %}
{% raw %}The Egyptian hieroglyphics form a system of picture-writing,{% endraw %}
{% raw %}in which concrete objects were originally expressed by pictures re¬{% endraw %}
{% raw %}presenting them; e.g.{% endraw %}
{% raw %}'Face' hr <@> 'Eye' yrt <2>{% endraw %}
{% raw %}'Moon' 'y'h s*=*^ 'Sun' r' 0{% endraw %}
{% raw %}'Pigeon' vr "55^ 'Plough' hb' ^N^{% endraw %}
{% raw %}Abstract ideas and verbs were represented on the same principle{% endraw %}
{% raw %}by the use of pictures of objects suggesting in some sort the idea{% endraw %}
{% raw %}to be expressed. Thus the idea 'to rule' hk' was expressed by the{% endraw %}
{% raw %}picture of a sceptre |, 'south' rs'hy a lily *!-<•, the botanical emblem{% endraw %}
{% raw %}of Upper Egypt, 'to find' gmy by an ibis feeding (^^, etc.{% endraw %}
