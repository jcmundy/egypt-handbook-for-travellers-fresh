{% raw %}During the second century after Christ Egyptian magical formulae{% endraw %}
{% raw %}were frequently written in Greek characters; and after the intro¬{% endraw %}
{% raw %}duction of Christianity it became the universal custom to write the{% endraw %}
{% raw %}Egyptian translations of the Scriptures in the simpler Greek letters{% endraw %}
{% raw %}instead of in the inconvenient hieroglyphics, which were at the{% endraw %}
{% raw %}same time more difficult to learn. But as the Greek alphabet was{% endraw %}
{% raw %}not adequate to represent all the Egyptian sounds (e.g. sh, ; kh, etc){% endraw %}
{% raw %}seven supplementary symbols t were borrowed from the demotic.{% endraw %}
{% raw %}Thus arose the Coptic Writing of the Egyptian Christians.{% endraw %}
{% raw %}The use of hieroglyphics extended beyond the borders of Egypt,{% endraw %}
{% raw %}especially into Nubia, where they were employed in the temples{% endraw %}
{% raw %}built by the Pharaohs. And even after the Nubian - Ethiopian{% endraw %}
{% raw %}kingdom became independent of Egypt in the 8th cent. B.C.,{% endraw %}
{% raw %}hieroglyphics still continued to be used there. At first, however,{% endraw %}
{% raw %}only inscriptions in the Egyptian language were thus written; some{% endraw %}
{% raw %}time elapsed before hieroglyphics were adapted to the native lan¬{% endraw %}
{% raw %}guage, which was allied to the modern Nubian tongue. In the{% endraw %}
{% raw %}course of this adaptation various formal modifications took place,{% endraw %}
{% raw %}resulting in an Ethiopian Hieroglyphic System, which has not as{% endraw %}
{% raw %}yet been fully deciphered. In the post-Christian era an Ethiopian{% endraw %}
{% raw %}Cursive Style, apparently based on the demotic, was also developed.{% endraw %}
{% raw %}This also has not yet been deciphered.{% endraw %}
