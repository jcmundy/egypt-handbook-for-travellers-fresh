{% raw %}9. Medinet Habu.{% endraw %}
{% raw %}Visitors to Medinet Habu should consult the 'Notice explicative des{% endraw %}
{% raw %}mines de Me'dinet Habou', by Georges Daressy (Cairo, 1897), a detailed de¬{% endraw %}
{% raw %}scription issued by the Archaeological Administration. — The Inspector{% endraw %}
{% raw %}General of the Antiquities lives close by the temple.{% endraw %}
{% raw %}About 1 M. from Der el-Medineh and Y2 M. from the Memnon{% endraw %}
{% raw %}colossi (p. 305), in a conspicuous situation, lies the southernmost{% endraw %}
{% raw %}temple-group of the Theban Necropolis. This bears the name of{% endraw %}
{% raw %}Medinet Habu, a Christian village which arose around and even{% endraw %}
{% raw %}within the ancient sanctuary as early as the 5th cent., and of which{% endraw %}
{% raw %}considerable traces still remain.{% endraw %}
{% raw %}The entire edifice may be divided into two easily distinguished{% endraw %}
{% raw %}portions. One of these is the small temple of the 18th Dynasty{% endraw %}
{% raw %}(p. 303), afterwards enlarged. The other is the main temple of{% endraw %}
{% raw %}Eamses III. (p. 299), which was enclosed by a crenelated wall,{% endraw %}
{% raw %}23 ft. high. We pass in front of the pylon and other buildings in{% endraw %}
{% raw %}front of the small temple, and visit first the structure of Eamses.{% endraw %}
{% raw %}We pass through the outer wall, by means of a Gate (PL A),{% endraw %}
{% raw %}about 3 ft. wide, which is flanked by two small Porter's Lodges. Be¬{% endraw %}
{% raw %}yond these we are confronted by the remarkable edifice, resembling{% endraw %}
{% raw %}a castle and known as the Pavilion of Eamses III., which forms{% endraw %}
{% raw %}the entrance to the precincts of the temple instead of the usual{% endraw %}
{% raw %}massive pylon and portals of stone. This structure reproduces the{% endraw %}
{% raw %}shape of the Syrian fortresses which occur so often in reliefs (e.g. on{% endraw %}
{% raw %}the first pylon of the Eamesseum, p. 284). Two tall towers (PL 6, c),{% endraw %}
{% raw %}with almost imperceptibly sloping walls and with their E. sides resting{% endraw %}
{% raw %}upon a battering foundation-wall, enclose a narrow court, gradually{% endraw %}
{% raw %}contracting towards the back, where a door (PL a) has been formed{% endraw %}
{% raw %}in the central erection joining the towers. This edifice, which is{% endraw %}
{% raw %}built of hewn stone, is only the inner kernel of the ancient building;{% endraw %}
{% raw %}it was originally adjoined on each side by brick structures of the{% endraw %}
{% raw %}full breadth of the present facade. These, however, were destroyed{% endraw %}
