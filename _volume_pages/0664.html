{% raw %}GEEF-HUSEN.{% endraw %}
{% raw %}From Kalabsheh{% endraw %}
{% raw %}Bock Temple of Gerf-Husen. The Egyptian name of this temple{% endraw %}
{% raw %}was Per-Ptah, the 'House of Ptah'. It was founded in the reign of{% endraw %}
{% raw %}Eamses II. by Setaw, at that time governor of Ethiopia, and it was{% endraw %}
{% raw %}dedicated to Ptah of Memphis and his fellow-gods. In front of{% endraw %}
{% raw %}the rock-temple proper lay a{% endraw %}
{% raw %}quadrangular Court (PL A),{% endraw %}
{% raw %}which was surrounded on{% endraw %}
{% raw %}three sides by covered colon¬{% endraw %}
{% raw %}nades. The E. colonnade had{% endraw %}
{% raw %}lotus-columns, while the{% endraw %}
{% raw %}others were supported by{% endraw %}
{% raw %}pillars, against which stood{% endraw %}
{% raw %}colossal figures of Eamses II.{% endraw %}
{% raw %}Two of the columns and five of{% endraw %}
{% raw %}the pillars are still standing.{% endraw %}
{% raw %}The W. side of the court is{% endraw %}
{% raw %}bounded by the rock-fac,ade{% endraw %}
{% raw %}of the temple, hewn to imi¬{% endraw %}
{% raw %}tate a pylon and decorated in{% endraw %}
{% raw %}a corresponding style. On the{% endraw %}
{% raw %}left side of the portal is a re¬{% endraw %}
{% raw %}lief of Eamses offering fresh{% endraw %}
{% raw %}vegetables to Ptah. Passing{% endraw %}
{% raw %}through this portal, we enter{% endraw %}
{% raw %}a large Hall (PL B), 45 ft.{% endraw %}
{% raw %}square, hewn out of the rock.{% endraw %}
{% raw %}The ceiling is supported by{% endraw %}
{% raw %}six pillars, 28 ft. high, against{% endraw %}
{% raw %}which are statues of the king{% endraw %}
{% raw %}like those in the court. On{% endraw %}
{% raw %}each side of the hall are{% endraw %}
{% raw %}four recesses, each with the{% endraw %}
{% raw %}king, in the guise of a god{% endraw %}
{% raw %}under various titles, standing{% endraw %}
{% raw %}between two deities.{% endraw %}
{% raw %}In the Recesses on the Left (S.) Side (from left to right). 1. The{% endraw %}
{% raw %}king between Ammon-Re and Mut; 2. Between Horus, lord of Beki, and{% endraw %}
{% raw %}Horus, lord of Beheni; 3. Between Ptah-Tatenen and the cow-headed Hathor.{% endraw %}
{% raw %}4. Between Ptah and Sekhmet.{% endraw %}
{% raw %}In the Recesses on the Right (N.) Side (from left to right): 1. The{% endraw %}
{% raw %}king between Khnum and Anukis ; 2. between Nefertem and Satet; 3. be¬{% endraw %}
{% raw %}tween Horus, lord of Mem, and Isis; 4. between Harmachis and Ews-os.{% endraw %}
{% raw %}The reliefs on the walls of the hall, showing the king before various{% endraw %}
{% raw %}deities, are unimportant.{% endraw %}
{% raw %}The following Anteroom (PL C), about 36 ft. wide though only{% endraw %}
{% raw %}17 ft. deep, is entered by a small door, on the left side of which is{% endraw %}
{% raw %}the king before Ptah. The ceiling is supported by two square pillars.{% endraw %}
{% raw %}The walls and pillars are embellished with representations of the{% endraw %}
{% raw %}king in presence of various gods (including the deified Eamses).{% endraw %}
